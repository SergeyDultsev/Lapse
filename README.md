## Установка
Для установки и запуска проекта, необходим [NodeJS](https://nodejs.org/) v20+.

```bash
# Клонирование
git clone https://github.com/yourusername/Lapse.git
cd Lapse

# Запуск докера
docker compose up -d --build
```
